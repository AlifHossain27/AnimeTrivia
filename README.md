# **Installation**

### **Clone this repository :**
```
$ git clone https://github.com/AlifHossain27/AnimeTrivia.git
```

### **Go into the repository :**
```
$ cd AnimeTrivia
```

### **Install dependencies :**
```
$ pip install -r requirements.txt
```

### **Run the app :**
```
$ python trivia.py
```
